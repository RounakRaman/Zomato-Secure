# Zomato × DPDP — "Consent, Clearly"

A clickable prototype of a **DPDP-aligned consent & data-rights experience** for Zomato, built in Streamlit.
It walks the end-to-end happy path and, in the sidebar, maps every screen to the specific provision of India's
**Digital Personal Data Protection Act, 2023** and the **DPDP Rules, 2025**.

## What it demonstrates

- **Itemised, unbundled consent** — essentials grouped and labelled; every optional purpose separate and **OFF by default** (DPDP Rule 3)
- **Notice in your language** — English / हिन्दी toggle (+ note on all 22 scheduled languages)
- **Two taps from home** — Profile → Privacy & Data
- **Withdrawal parity** — turn any consent off in one tap; essentials show an **honest consequence**, not a dark pattern
- **Data-principal rights** — download (access), correct, erase, nominate (DPDP Rule 14)
- **Grievance redressal** — DPO channel with a published SLA (DPDP Rule 13)
- **Children's data flow** — age assurance → verifiable parental consent → protected mode with no tracking or targeted ads (DPDP Rule 10)
